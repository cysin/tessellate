# GV Board Cutting System - Simple Design V2

**Version:** 2.0
**Purpose:** High-level design with single-page layout and Excel upload support

---

## System Purpose

Optimize cutting of furniture parts from raw boards to minimize material waste.

**Input:** List of parts to cut + board specifications (manual entry OR Excel upload)
**Output:** Visual cutting plans showing how to cut each board
**Layout:** Single page - input and output both visible for easy editing
