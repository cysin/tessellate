APPLICATION OVERVIEW
================================================================================

Name:        GV (开板系统) - Board Cutting Optimization System
Purpose:     Optimize furniture panel cutting to minimize material waste
Industry:    Furniture manufacturing / Interior design
Algorithm:   First-Fit Decreasing (FFD) heuristic with guillotine cuts
Input:       Product specs (dimensions, quantity, grain direction)
Output:      Visual cutting diagrams with material utilization stats

Technology Stack:
  - Java (Backend)
  - Struts 2.3.x (MVC Framework)
  - Spring 3.0.6 (IoC - minimal usage)
  - JSP + jQuery (Frontend)
  - JSON (Data exchange)

Key Features:
  1. Multi-product optimization
  2. Grain direction constraints
  3. Leftover material tracking and reuse
  4. Multiple board sizes support (4×8, 5×8, 4×9, 5×9, 4×10, 5×10 feet)
  5. Saw kerf compensation
  6. Utilization threshold filtering
  7. Visual cutting diagram generation
  8. Print functionality
