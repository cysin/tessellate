HOW TO USE
================================================================================

1. Open: http://localhost:8089/GV/input.jsp (use --noproxy "*" with curl)

2. For each product line above:
   - Click "添加" (Add) button to create new row
   - Fill in: 编号, 名称, 长, 宽, 厚, 花色, 数量
   - Select 纹理方向: 单向 or 混合

3. Set board configuration at bottom

4. Optional: For Scenario 4, check "库存余料优先" and add leftover data

5. Click "确认完成" then "确认开板" to generate cutting plan

6. View results: cutting diagrams, utilization rate, leftover pieces


FIELD MEANINGS:
--------------
编号 (Code)        - Product identifier
名称 (Name)        - Product name/description
长 (Length)        - Length in millimeters
宽 (Width)         - Width in millimeters  
厚 (Thickness)     - Thickness in millimeters
花色 (Color)       - Wood color/pattern
数量 (Quantity)    - Number of pieces needed
纹理 (Grain)       - 单向=Directional (no rotation), 混合=Mixed (can rotate)


TIPS:
----
- Start with Scenario 3 (Minimal Test) - easiest to enter
- All dimensions are in millimeters
- Utilization rate 0.75-0.80 is typical (75-80%)
- Saw kerf 3mm is standard for most saws
- Use larger boards (5×9, 5×10) for less waste
